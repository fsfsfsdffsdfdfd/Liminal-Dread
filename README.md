# Liminal Dread Modpack

Welcome to the official issue tracker for the **Liminal Dread** modpack!

This repository is dedicated to tracking bugs, issues, and feature requests for the modpack. If you've encountered a problem or have a suggestion, please report it here.

## 📋 How to Report an Issue

### Before Reporting:
- Check if your issue has already been reported
- Make sure you're using the latest version of the modpack
- Try removing recently added mods to identify the culprit

### When Reporting:
1. Use the **Bug Report** template for crashes and problems
2. Use the **Feature Request** template for suggestions
3. Include as much detail as possible:
   - **What happened?** (describe the problem)
   - **When did it happen?** (what were you doing?)
   - **Log files** (crash logs are super helpful!)
   - **Modpack version** you're using
   - **Java version** and **RAM allocated** to Minecraft

### Where to Get Help:
- 📧 **Email**: (LiminalDread@outlook.com)

## 🎯 Issue Labels

Issues are organized by type:
- **bug**: Something isn't working
- **feature-request**: Suggestion for a new feature
- **balance**: Gameplay balance concerns
- **documentation**: Website or README updates
- **duplicate**: Already reported issue
- **invalid**: Not actually a bug or outside scope

## ⚠️ Important Notes

- Please be respectful and constructive in your feedback
- Duplicate reports will be closed (but you'll get credit if you find one first!)
- I'll do my best to respond, but we may not fix everything immediately

## 🙏 Thanks!

Thanks for using Liminal Dread and helping make it better!
